72 

73 This class represents a contextual compositor that modifies the behavior of the printer. 

74 It provides various options for customizing the output format of the prompts within this context manager. 

75 

76 Attributes: 

77 _sep: The class default separator string is None, indicating inherit from the parent. 

78 _indexing: The class default indexing mode is empty indexing. 

79 _inc_indent: The class default indentation string is empty string. 

80 _new_indent: The class default new indentation string is None, indicating not overwrite the indent. 

81 _is_inline: The class default inline flag is False. 

82 _new_role: The class default role of the modifier is None, indicating not overwrite the role. 

83 """
